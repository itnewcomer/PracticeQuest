# PracticeQuest / おけいこクエスト

子供のおけいこ・宿題を楽しいクエストに変えるアプリ / Turn kids' daily practice into fun quests!

## Features / 機能

- 4 quest types: count, page, timer, stopwatch
- Morning bonus (2x stars before school)
- Bedtime penalty (reduced stars)
- Extra quests with 3x bonus after all clear
- Weekly schedule with lesson blocks & free time calc
- Stats with weekly chart, stopwatch trends, milestones
- Reward shop with time-based rewards (pause/resume timer)
- Parent settings: quests, lessons, rewards, school times
- Japanese / English support

## Science-Based Design

- Token Economy (Kazdin) — 80+ years of research
- Variable Ratio Schedule (Skinner)
- Self-Determination Theory (Deci & Ryan)
- Habit Formation (Lally et al., 2010) — 66 days
- Morning Learning Effect (Synchrony Effect)
